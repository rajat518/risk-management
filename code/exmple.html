Human Resources (HRS)

Overview

 

The HR programme is run in partnership with IS Applications. This is a new programme within the SASG portfolio. Previously, HR had a joint programme with Payroll and Pensions in the CSG portfolio. This programme will cover the University's annual HESA return projects, any eRecruitment projects and those projects which will be required to deliver the University's People Plan.

In 14/15, the current plan includes:

  • Completing HRS082 - HESA 13/14 return
  • Commencing HESA 14/15 return
  • Enhancing Employment: This initiative will progress the system changes required to support any agreed Policy or process changes as a result of the on-going review of Hourly Paid and Casual Employment policy and processes
  • eRecruitment Phase 2: The University is in the process of finalising revised Redeployment and Advertising Policies and it is intended that eRecruitment will facilitate part of the new processes. These changes along with the introduction of the Talent Pool functionality are to be rolled out by end 2014/15.In order to obtain enhancements as standard the software will likely undergo a substantial upgrade as part of this initiative and the impact to end users will need to be carefully planned.
  • Learning Management: This initiative will progress the recommendations from HR’s Learning Management Readiness and Business Case project. It is expected that this will entail procuring/developing and implementing Learning Management System(s) (including an Annual Review system) for all staff across the university and that this will be a multi-year project
